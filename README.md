# Metal Cardbot Battle Simulator

Metal Cardbot Battle Simulator is a turn-based CLI battle game featuring characters from the Metal Cardbot franchise. Choose your fighter and battle against a random CPU opponent!

## Characters
There are 12 playable characters from Season 1:
Blue Cop, Mega Trucker, Mega Ambler, Phoenix Fire, Shadow X, Fleta Z, Dexter, Buster Gallon, Buffalo Crush, Wild Guardy, Black Hook, and Heavy Iron.

## How to use
1. Install dependencies
`pip install -r requirements.txt`

2. Run the game
`python3 main.py`

3. Choose your character from the roster and battle!
